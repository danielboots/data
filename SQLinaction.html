SQL in Action

A standard language for querying relational database management systems Provides commands for creating and modifying databases, as well as querying and modifying data SQL can be run as individual commands or scripts. It can be used through specialised tools, or from any programming language

SELECT

A statement for querying data from a relational database. Retrieves data from a specified list of columns in a table

example 

SELECT column1, column2, ...
            FROM table_name;
