SQL in Action

A standard language for querying relational database management systems Provides commands for creating and modifying databases, as well as querying and modifying data SQL can be run as individual commands or scripts. It can be used through specialised tools, or from any programming language

SELECT

A statement for querying data from a relational database. Retrieves data from a specified list of columns in a table

example 

SELECT column1, column2, ...
            FROM table_name;

example

  • SELECT Name from Artist;

    • will return all rows names etc

  • select FirstName,LastName from Customer;
  • SELECT * FROM Track;
  • SELECT Name FROM MediaType;

WHERE Clause

filtering the results using the WHERE clause

example

  • SELECT * FROM Track WHERE Composer = 'U2';
  • SELECT * FROM Album WHERE AlbumID =232;

    • JOIN

    GROUP BY Clause

    Often, what we really want to do is group our data into a number of subsets and use the aggregate functions on each of those. For example, counting how many tracks use each of the five different media types. SQL provides a group by feature that allows us to do this. Here, we will group by the AlbumID in the track table and show the number of rows in each group.

    SELECT AlbumID, COUNT(*) FROM Track
    GROUP BY AlbumId;
    SELECT Album.Title,COUNT(*) FROM Track 
    INNER JOIN Album ON Track.AlbumId = Album.AlbumId
    GROUP BY Track.AlbumId;
