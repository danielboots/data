NOTE: Special Characters in MongoDB Atlas Usernames and Passwords. After creating your cluster, we add a SCRAM username and password at 2:17 in this video. Do not use any non-alphanumeric characters (e.g.: ?, %,@ or &) in your username or password! If you do use any of these "special" characters then, due to a bug in MongoDB's URI parser, you will not be able to connect to your database using Python later in the lesson. We recommend using a combination of upper- and lower-case letters, and numbers for your database password.

finding records

standard command: coll.find()

  2. coll.find({gender: "f"})
  3. coll.find({gender: "f", nationality: "british"})
  4. coll.find({gender: "f", $or: [{nationality: "british"}, {nationality: "irish"}]});

    5. sorting:

  5. coll.find({gender: "f", $or: [{nationality: "british"}, {nationality: "irish"}]}).sort({nationality: 1});

update and delete data

Running commands (update and remove) from the command line to update your MongoDB Database. The update command allows you to update existing database entries. The remove command allows you to delete existing database entries.

updating records

    update only the first record

  1. coll.update({nationality: "irish"}, {$set: {hair_color: "blue"}});

    2. update all records

  2. coll.update({nationality: "irish"}, {$set: {hair_color: "purple"}}, {multi: true});

Delete records

  1. coll.remove({first: "kate", last: "bush"});
